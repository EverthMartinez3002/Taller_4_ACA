    API de Cupcakes#

    La API de cupcakes permite gestionar el catálogo completo de productos, incluyendo recetas, ingredientes, precios, disponibilidad y personalizaciones.

    Base URL#

    https://api.cupcakeapi.com/v1/cupcakes

    🧁 Catálogo de Cupcakes#

    Listar Cupcakes#

    Obtiene una lista paginada de cupcakes con filtros opcionales.

    GET /cupcakes

    Parámetros de Query:

    Parámetro Tipo Descripción Ejemplo
    page int Número de página (default: 1) ?page=2
    limit int Items por página (default: 20, max: 100) ?limit=50
    category string Filtrar por categoría ?category=premium
    available boolean Solo productos disponibles ?available=true
    search string Buscar en nombre/descripción ?search=chocolate
    sort string Ordenar por campo ?sort=price
    order string Dirección (asc/desc) ?order=desc
    min_price float Precio mínimo ?min_price=2.00
    max_price float Precio máximo ?max_price=5.00
    allergens string Excluir alérgenos ?allergens=nuts,dairy
    dietary string Filtrar por dieta ?dietary=vegan,gluten_free
    difficulty string Nivel de dificultad ?difficulty=easy
    prep_time int Tiempo máximo de prep. ?prep_time=30

    Ejemplo de Petición: 

    curl -H "Authorization: Bearer YOUR_API_KEY" \
     "https://api.cupcakeapi.com/v1/cupcakes?category=premium&available=true&sort=popularity&order=desc&limit=10"

    Actualizar Cupcake#

    Modifica un cupcake existente.

    PUT /cupcakes/{id}
Authorization: Bearer JWT_TOKEN
Content-Type: application/json

    Actualizar Stock#

    Actualiza únicamente el stock de un cupcake.

    PATCH /cupcakes/{id}/stock
Authorization: Bearer JWT_TOKEN
Content-Type: application/json

    Cuerpo de la Petición: 

    {
    "stock": 50,
    "operation": "set", // "set", "add", "subtract"
    "reason": "Restock semanal",
    "notify_if_low": true,
    "low_stock_threshold": 5
}

    Actualizar Precio#

    Modifica el precio de un cupcake.

    PATCH /cupcakes/{id}/price
Authorization: Bearer JWT_TOKEN
Content-Type: application/json

    Cuerpo de la Petición: 

    {
    "price": 4.25,
    "original_price": 5.00,
    "discount_percentage": 15,
    "effective_date": "2024-01-25T00:00:00Z",
    "end_date": "2024-02-14T23:59:59Z",
    "reason": "Promoción San Valentín"
}

    Cambiar Disponibilidad#

    Activa o desactiva un cupcake.

    PATCH /cupcakes/{id}/availability
Authorization: Bearer JWT_TOKEN
Content-Type: application/json

    Cuerpo de la Petición: 

    {
    "available": false,
    "reason": "Temporarily out of ingredients",
    "estimated_return": "2024-01-25T10:00:00Z"
}

    Eliminar Cupcake#

    Elimina un cupcake del catálogo (soft delete).

    DELETE /cupcakes/{id}
Authorization: Bearer JWT_TOKEN

    🖼️ Gestión de Imágenes#

    Subir Imagen#

    Sube una nueva imagen para un cupcake.

    POST /cupcakes/{id}/images
Authorization: Bearer JWT_TOKEN
Content-Type: multipart/form-data

    Cuerpo de la Petición: 

    Content-Disposition: form-data; name="image"; filename="cupcake.jpg"
Content-Type: image/jpeg

[binary image data]

Content-Disposition: form-data; name="alt_text"
Vista frontal del cupcake

Content-Disposition: form-data; name="is_primary"
false

    Reordenar Imágenes#

    Cambia el orden de las imágenes de un cupcake.

    PUT /cupcakes/{id}/images/order
Authorization: Bearer JWT_TOKEN
Content-Type: application/json

    Cuerpo de la Petición: 

    {
    "image_order": [3, 1, 2, 4]
}

    Eliminar Imagen#

    Elimina una imagen específica.

    DELETE /cupcakes/{id}/images/{image_id}
Authorization: Bearer JWT_TOKEN

    Crear Review#

    Permite a un usuario crear una reseña de un cupcake.

    POST /cupcakes/{id}/reviews
Authorization: Bearer JWT_TOKEN
Content-Type: application/json

    Cuerpo de la Petición: 

    {
    "rating": 5,
    "title": "¡Excelente cupcake!",
    "comment": "Superó mis expectativas. Perfecto balance de sabores y muy buena presentación.",
    "recommend": true
}

    Marcar Review como Útil#

    Vota si una reseña es útil o no.

    POST /cupcakes/{cupcake_id}/reviews/{review_id}/helpful
Authorization: Bearer JWT_TOKEN
Content-Type: application/json

    Cuerpo de la Petición: 

    {
    "helpful": true
}

    🔍 Búsqueda Avanzada#

    Búsqueda por Texto#

    Busca cupcakes por nombre, descripción o ingredientes.

    GET /cupcakes/search?q={query}

    Parámetros de Query: - q (string): Término de búsqueda - fields (string): Campos donde buscar (name, description, ingredients, tags) - Todos los filtros de la lista principal

    Ejemplo: 

    curl "https://api.cupcakeapi.com/v1/cupcakes/search?q=chocolate&fields=name,description&category=premium"

    Sugerencias de Búsqueda#

    Obtiene sugerencias de búsqueda autocompletada.

    GET /cupcakes/search/suggestions?q={partial_query}

    📊 Analytics de Cupcakes#

    Cupcakes Populares#

    Obtiene los cupcakes más populares.

    GET /cupcakes/popular
Authorization: Bearer JWT_TOKEN

    Parámetros de Query: - period (string): Período (day, week, month, year) - limit (int): Cantidad de resultados

    Tendencias#

    Obtiene cupcakes con tendencias de popularidad.

    GET /cupcakes/trending
Authorization: Bearer JWT_TOKEN

    Códigos de Error Específicos#

    Código Descripción Solución
    CUPCAKE_NOT_FOUND Cupcake no encontrado Verificar ID del cupcake
    CUPCAKE_OUT_OF_STOCK Sin stock disponible Seleccionar otro producto o esperar restock
    INVALID_CUSTOMIZATION Personalización inválida Verificar opciones disponibles
    PRICE_OUT_OF_RANGE Precio fuera del rango permitido Ajustar filtros de precio
    CATEGORY_NOT_FOUND Categoría no existe Usar categorías válidas
    DUPLICATE_CUPCAKE_NAME Nombre ya existe Usar nombre único
    INVALID_INGREDIENT Ingrediente no válido Usar ingredientes del catálogo
    IMAGE_UPLOAD_FAILED Error al subir imagen Verificar formato y tamaño

    La API de cupcakes proporciona todas las funcionalidades necesarias para gestionar un catálogo completo de productos, desde la navegación básica hasta funciones administrativas avanzadas y análisis de popularidad.
