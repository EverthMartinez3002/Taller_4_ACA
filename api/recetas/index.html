    API de Recetas#

    La API de recetas permite gestionar recetas de cupcakes, ingredientes, instrucciones de preparación y técnicas de decoración para uso interno y compartido.

    Base URL#

    https://api.cupcakeapi.com/v1/recipes

    🍰 Gestión de Recetas#

    Listar Recetas#

    Obtiene una lista de recetas disponibles.

    GET /recipes
Authorization: Bearer JWT_TOKEN

    Parámetros de Query: - category (string): Filtrar por categoría (masa, cobertura, relleno, decoracion) - difficulty (string): Nivel de dificultad (easy, medium, hard, expert) - prep_time (int): Tiempo máximo de preparación en minutos - dietary (string): Filtros dietéticos (vegan, gluten_free, sugar_free, keto) - allergens (string): Excluir alérgenos (nuts, dairy, eggs, gluten) - equipment (string): Filtrar por equipo requerido - search (string): Buscar en nombre o descripción - featured (boolean): Solo recetas destacadas - public (boolean): Solo recetas públicas

    Actualizar Receta#

    Modifica una receta existente.

    PUT /recipes/{id}
Authorization: Bearer JWT_TOKEN
Content-Type: application/json

    Versionar Receta#

    Crea una nueva versión de una receta existente.

    POST /recipes/{id}/versions
Authorization: Bearer JWT_TOKEN
Content-Type: application/json

    Cuerpo de la Petición: 

    {
    "version_notes": "Ajuste de temperatura de horneado para mejor textura",
    "changes": [
        {
            "type": "instruction_update",
            "step": 6,
            "old_value": "180°C por 18-20 minutos",
            "new_value": "175°C por 20-22 minutos"
        }
    ],
    "tested_by": "Chef Ana Rivera",
    "test_results": "Mejora significativa en textura y humedad"
}

    🥘 Gestión de Ingredientes#

    Listar Ingredientes#

    Obtiene el catálogo de ingredientes disponibles.

    GET /ingredients
Authorization: Bearer JWT_TOKEN

    Parámetros de Query: - category (string): Categoría de ingrediente - allergen (string): Filtrar por alérgeno - supplier (string): Proveedor específico - in_stock (boolean): Solo ingredientes en stock

    🔄 Conversión de Unidades#

    Convertir Cantidades#

    Convierte cantidades de ingredientes entre diferentes unidades.

    POST /recipes/convert
Authorization: Bearer JWT_TOKEN
Content-Type: application/json

    Cuerpo de la Petición: 

    {
    "ingredient": "harina de trigo",
    "from_amount": 240,
    "from_unit": "gramos",
    "to_unit": "tazas",
    "density": 125 // gramos por taza (opcional)
}

    Respuesta (200): 

    {
    "status": "success",
    "data": {
        "conversion": {
            "original": {
                "amount": 240,
                "unit": "gramos"
            },
            "converted": {
                "amount": 1.92,
                "unit": "tazas"
            },
            "ingredient": "harina de trigo",
            "density_used": 125,
            "precision": "approximate"
        }
    }
}

    Escalar Receta#

    Ajusta las cantidades de una receta para diferentes porciones.

    POST /recipes/{id}/scale
Authorization: Bearer JWT_TOKEN
Content-Type: application/json

    Cuerpo de la Petición: 

    {
    "target_servings": 24,
    "original_servings": 12,
    "round_to_practical": true
}

    📊 Analytics de Recetas#

    Popularidad de Recetas#

    Obtiene estadísticas de uso y popularidad de recetas.

    GET /recipes/analytics/popularity
Authorization: Bearer JWT_TOKEN

    Parámetros de Query: - period (string): Período de análisis - chef_id (int): Filtrar por chef - category (string): Filtrar por categoría

    Costo de Recetas#

    🎓 Sistema de Certificaciones#

    Certificar Competencia#

    Certifica que un chef domina una receta específica.

    POST /recipes/{id}/certifications
Authorization: Bearer JWT_TOKEN
Content-Type: application/json

    Cuerpo de la Petición: 

    {
    "chef_id": 456,
    "evaluator_id": 123,
    "test_date": "2024-01-20T10:00:00Z",
    "score": 95,
    "competencies": {
        "technique": 98,
        "timing": 92,
        "presentation": 95,
        "consistency": 90
    },
    "notes": "Excelente dominio de la técnica, resultados muy consistentes",
    "certified": true,
    "certification_level": "advanced"
}

    Códigos de Error Específicos#

    Código Descripción Solución
    RECIPE_NOT_FOUND Receta no encontrada Verificar ID de la receta
    INGREDIENT_NOT_AVAILABLE Ingrediente no disponible Usar sustituto o esperar restock
    INVALID_SCALING_FACTOR Factor de escalado inválido Usar factor entre 0.1 y 10
    CONVERSION_NOT_SUPPORTED Conversión no soportada Usar unidades compatibles
    INSUFFICIENT_PERMISSIONS Permisos insuficientes Verificar rol del usuario
    RECIPE_IN_USE Receta en uso activo No se puede eliminar
    DUPLICATE_RECIPE_NAME Nombre duplicado Usar nombre único
    INVALID_INGREDIENT_COMBINATION Combinación inválida Verificar compatibilidad

    La API de recetas proporciona un sistema completo para gestionar conocimiento culinario, desde recetas básicas hasta análisis avanzados de costos y certificaciones de competencias.
